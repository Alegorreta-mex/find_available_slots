# Function Available Spots

## Important Notes
This is a project requested by a company to evaluate the technical skills of a candidate.
The Scope of the project is to resolve the problem presented in the file 'Availabel spots.docx', 
the document is not present on the file to avoid any kind of plagiarism or any compromise of the information. 

Approximate time to resolve the problem(you can see on the git history): 1 hour.

I let the solution in a time_complexity of O(n) because with the considerations try to resolve with a better time complexity 
is not worthy. You will need a lot of offices to have a problem with the time complexity of the solution.

## Description
The project is divides into two files:
- main.py : main file that contains the python script of the solution requested.
- test.py : file that contains the tests of the main.py file.

## How to run the project
To run the project you need to have python installed on your machine, the version used to develop this project was the 3.12.3.
We only use the standard libraries of python, so you don't need to install any other library.

To run the project you need to run the following command on the terminal:

```python main.py```

To run the tests you need to run the following command on the terminal:

```python test.py```
